This is not just a plugin, it symbolizes the hope and enthusiasm of an entire generation summed up in a post on wptavern by Jeff Chandler. 

== Description ==

When activated you will randomly see a lyric from <cite>Hello, Admin</cite> in the admin notices screen on every wp-admin page.

Please read Jeff’s article: http://wptavern.com/please-stop-abusing-wordpress-admin-notices
